# Модуль Оформление в 1 шаг + Яндекс.Быстрый заказ
Данный модуль будет очень удобен и практичен любому магазину на shop-script. Гибкие настройки и большой функционал позволит сделать оформление заказа в Вашем магазине легким и удобным. Внедренный в плагин "Яндекс.Быстрый заказ" позволит вашим покупателям заполнять необходимые поля всего за несколько кликов.

<h2>Возможности модуля:</h2>
	- Перерасчет корзины без перезагрузки окна;
	- Все цены считаются автоматически без перезагрузки страницы;
	- Возможность оформить заказ, введя ВСЕ данные или оформить БЫСТРЫЙ заказ;
	- Настраиваемые поля для разных способов оформления;
	- Внедренный плагин Яндекс.Быстрый заказ;
	- Возможность выставить к доп. полям свои значение от Яндекс.Быстрый заказ;
	- Ajax проверка формы перед подтверждением;
	- Возможность изменять характеристики товара прямо в корзине;
	- Возможность указать плательщика отличного от получателя;
	- Минимальные изменения в стандартной сборке движка;
	- Сохранены все возможности при оформлении заказ кроме:
        Google Checkout,
        PPExpress Checkout,

<h2>Установка</h2>

Установка модуля полностью автоматизирована и состоит она всего из нескольких пунктов. 

<b>Шаг 1</b>
    
	Залить файлы из архива в папку с webAsyst shop-script. 

<b>Шаг 2</b>
    
	Авторизуйтесь в админ-панели, затем перейдите по адресу http://домен/install.php. 
	Ознакомьтесь с условиями предоставления скриптов нажмите "Установить". 

<b>Шаг 3</b>
    
	Скрипт автоматически проверит наличие файлов модуля. При возникновении ошибки, 
	убедитесь что файлы находятся в нужной папке. Для продолжения нажмите "Продолжить". 

<b>Шаг 4</b>

	Скрипт автоматически внесет изменения в файлы webAsyst. При возникновении проблемы, 
	установите права записи на указанные файлы 777, затем измените их на первоначальные 
	права. Для продолжения нажмите "Продолжить". 

<b>Шаг 5</b>
	 
	Скрипт автоматически добавит записи в базу данных. Если часть записей уже находится 
	в базе данных, то они будут удалены и перезаписаны заново. Для продолжения нажмите 
	"Закончить установку". 

<b>Шаг 6</b>
	 
	Зайдите в админ панель в раздел "Дизайн" - "Языки и перевод" - Выберите ваш язык и
	нажмите "Редактировать перевод". Опуститесь в самый низ и нажмите "сохранить".

<b>Шаг 7</b>
	 
	Зайдите в админ панель в раздел "Настройки" - "Настройки" - "Корзина и заказы", 
	и проставьте необходимые настройки.

<b>Шаг 8</b>
	 
	Подключение Яндекс.Быстрый заказ 

	Для подключения "Яндекс.Быстрый заказ" перейдите по ссылке 
	http://partner.market.yandex.ru/delivery-registration.xml 
	и зарегистрируйте свой магазин в базе данных Яндекс, прописав в поле 
	"Адрес страницы перенаправления" путь http://Ваш домен/cart/ 
	(http://Ваш домен?ukey=cart для маазинов без ЧПУ).

	Внимание! В файле шаблона \published\SC\html\scripts\templates\frontend\onesteporder\main.html встроен фреймворк JQuery. 
    Если у вас уже подключен он, то удалите строчку в вышеупомянутом файле. 
    <script src="http://ajax.googleapis.com/ajax/libs/jquery/1.8.3/jquery.min.js"></script>

	Внимание! Если Вы будете использовать "информер" в качестве 
	завершающего диалогового окна, то в файле шаблона 
	\published\SC\html\scripts\templates\frontend\shopping_cart_info.html 
	удалите строчку class="{$checkout_class}" в ссылке.
